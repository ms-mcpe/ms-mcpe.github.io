Publica Crea y Disfruta de contenido publicado addons

Como funciona

Procesamiento Escribe el comando inicial >menu para ver la lista de comandos o si tienes dudas >help.
Contexto El uso del bot se basa en publicar addons o contenido que otros puedan descargar creando perfiles de jugadores existentes, que puedan interactuar como cuentas dentro de la plataforma del bot.
Privacidad Diceñado para minimizar datos sensibles; revisa las Politicas de la plataforma para detalles.

Como se usa

1. Invoca Escribe el comando o mensaje de Activacion en el grupo o chat privado donde esta integrado.
2. Busca Escribe Brevemente el comando >busd-(bus) para buscar algun contenido. recuerda remplazar (bus) por el contenido que buscas ejemplo: >busd-minecraft
3. Recibe Obten la respuesta, enlace o accion sugerida; si necesitas mas detalle, pide aclaracion.
